# Commands Reference

Every slash command in Amber, grouped by source file.

All commands use `app_commands.allowed_contexts(guilds=True, dms=True, private_channels=True)` and `allowed_installs(guilds=True, users=True)` unless marked **guild-only**.

---

## `commands/reactions.py` — `/do`, `/look`

The reaction GIF system. See [Reactions System](reactions-system.md) for full detail.

| Command | Description |
|---|---|
| `/do [action] [user?] [everyone?]` | Perform an anime action GIF toward a user or everyone |
| `/look [reaction]` | Express a solo reaction GIF |

**`/do` actions:** hug, kiss, pat, poke, cuddle, bite, kick, punch, feed, highfive, dance, sleep, cry, smile, think, wave, laugh, yeet, facepalm, baka, nom, shoot, run, stare, thumbsup

**`/look` reactions:** blush, shrug, yawn, angry, bored, happy, nope, smug, lurk, pout, nod

## `commands/melody.py` — `/melody`

| Command | Description |
|---|---|
| `/melody` | Opens a mode picker (Notes Mode or Beats Mode), then a modal for input |

Notes Mode: comma-separated notes like `C4 E4 G4` or `E4 1.0, G4 0.5`  
Beats Mode: pattern string like `_.-_..` where `_=0.6s`, `-=0.4s`, `.=0.2s`

Output: a `.wav` file generated by `wavesynth`, sent as an attachment.

## `commands/banking.py` — Economy & Profile

| Command | Description |
|---|---|
| `/money daily` | Claim 100–200 dabloons + 40–80 XP (24h cooldown) |
| `/money give [user] [amount]` | Transfer dabloons to another registered user |
| `/money balance` | Check your dabloon balance |
| `/money leaderboard` | Top 10 users by dabloon balance |
| `/profile` | Show your profile embed with bio, balance, action stats, and customization buttons |
| `/setbio [bio]` | Set your profile bio directly (also available via profile buttons) |
| `/level` | Check your level and XP to next level |

`/money give` requires the receiver to be registered. The sender is auto-registered if needed.

XP is also awarded passively per message via `message_xp_handler()` — random 5–25 XP adjusted by message length.

## `commands/minigames.py` — `/games`

| Command | Description |
|---|---|
| `/games duck_clicker` | Click the 🦆 button to accumulate clicks; earn 2 dabloons per 5 clicks |
| `/games tic_tac_toe [difficulty]` | Play against minimax AI; costs 2/4/8 dabloons, wins pay 4/8/16 |

**Duck Clicker upgrades** (from shop): Auto Clicker (auto-click every 60s), Double Points (4 dabloons per 5 clicks instead of 2).

**Tic Tac Toe difficulty levels:**

| Difficulty | Cost | Win Reward | AI Random Chance |
|---|---|---|---|
| easy | 2 | 4 | 30% random moves |
| medium | 4 | 8 | 25% random moves |
| hard | 8 | 16 | 10% random moves |

Both games track quest progress (`duck_clicks`, `ttt_win`).

## Non-Slash Commands (message events)

These are triggered by message content, not slash commands.

| Trigger | Behavior |
|---|---|
| Reply to a message with `4k` | Generates a quote card image of the replied message |
| Reply to a message with `pin` | Pins the replied message |
| Reply to a message with `unpin` | Unpins the replied message |
| Any message (non-bot) | Awards XP, tracks quest progress, updates pet inactivity timer |
| `/ping` | Returns bot latency in ms |
